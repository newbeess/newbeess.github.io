Learning Markdown

Learning Markdown

  1. 斜体 和 加粗
  2. 标题
  3. 链接
  4. 图片
  5. 引用
  6. 列表
  7. 段落
  8. 总结

###1.斜体 和 加粗 (Italics and Bold）

  • 斜体所需要的符号是些划线(underscore): _some words_
  • 粗体所需要的符号是星号: **some words**
    当然，你也可以同时设置字体为斜体和加粗: **_some words_**

###2. 标题(Headers)

  • 标题一共有六个层次，
    • # 一级标题
    • ## 二级标题
    • ### 三级标题
    • #### 四级标题
    • ##### 五级标题
    • ###### 六级标题

###3.链接(Links)

  • inline link: [newbeess] (https://github.com/newbeess)
    例如：[google](www.google.com)

  • reference link: [newbeess][tag]
    [tag]: https://github.com/newbeess

inline link 和 reference link 适用于不同的场景同样,前者适合文章的链接比较少的情境,后者适用于文章链接相对较多的情境,当然,个人的喜好也是决定你如何选择的原因.
你可以在链接词语上设置斜体活着粗体

###4.图片(Images)

  • inline image
    例如: ![newbeess](http://octodex.github.com/images/octdrey-catburn.jpg)
  • reference image
    例如: ![first image][tag1] 
[tag1]: http://octodex.github.com/images/founding-father.jpg  
 图片的加入模式和链接很相似,只需在[]前加入一个感叹号 !

###5.引用 (Blockquotes)

  • 如果引用一段只用一个“>”
  • 如果饮用多段使用多个“>”

###6.列表 (Lists)

  • 无序列表:
    * this one
    * this one
    * this one
  • 有序列表:
    1. first one
    2. second one
    3. third one
  • 列表嵌套:
    * outside list
    space space * inside list
    space space * inside list
    * outside list
    * outside list
    ###7. 段落(Paragraphs)
  • hard break 在两段之间插入一个空行
  • soft break 在每段的结尾插入两个空格
